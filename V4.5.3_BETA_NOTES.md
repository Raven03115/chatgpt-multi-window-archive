# v4.5.3 Beta — Performance Optimization

This branch-only beta keeps the v4.5.2 sidebar overlay and routing fixes, while applying low-risk performance changes:

- relevant-node, debounced pane chrome detection instead of full scans on every DOM mutation
- shape-state mutation filtering and a slower safety fallback
- duplicate `BrowserWindow.setShape()` suppression
- duplicate overlay bounds update suppression
- incremental active-pane border updates
- staggered ChatGPT pane loading at 160 ms intervals
- removal of duplicate navigation state assignment and hot-path dialog logging

No renderer freezing, pane destruction, or background-generation suspension is introduced.
